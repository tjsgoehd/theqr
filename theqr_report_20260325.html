1. 카드뉴스 기획안 (6 Slide)

핵심 카피와 비주얼 전략을 모두 담았습니다.

총 6페이지 구성
SLIDE 01: HOOKING

"홈페이지 제작, 아직도 200만 원씩 내시나요?"

비주얼: 영수증이 찢어지는 효과와 더큐알 로고 대비

SLIDE 2-3: SOLUTION

"복잡한 기능은 빼고, 매출에 필요한 것만 담았습니다."

비주얼: 복잡한 관리자 페이지 vs 심플한 더큐알 템플릿

SLIDE 4-5: BENEFITS

"단 2일 만에 완성, 모바일 최적화는 기본"

비주얼: 스마트폰에서 부드럽게 작동하는 사이트 시연

SLIDE 06: CTA

"선착순 10명 출시 기념 특별 할인가 문의하기"

비주얼: 프로필 링크 클릭 유도 화살표

2. 릴스 / 숏폼 시나리오

영상 제작 시 핵심 타임라인 가이드

타입 A: 3초 반전 (Before/After)

  • (0-3s) 투박한 네이버 블로그 화면이나 오래된 홈페이지를 보여주며 '한숨 쉬는 사장님'
  • (3-12s) 화면 전환 효과와 함께 **더큐알 템플릿**으로 세련되게 바뀐 화면 3~4개 교차 편집
  • (12-15s) "홈페이지, 이제 퀄리티로 승부하세요." 자막 노출

타입 B: 정보 전달 (Talking Head)

  • (0-5s) "홈페이지 제작비 아끼는 가장 확실한 방법, 알려드릴까요?"
  • (5-15s) 실속형 템플릿의 장점 3가지 자막으로 빠르게 노출 (저렴한 비용 / 빠른 제작 / 쉬운 관리)
  • (15-20s) "더큐알 프로필 링크에서 샘플 확인하세요!"
